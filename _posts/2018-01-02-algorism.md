## 문제

Get the number n to return the reversed sequence from n to 1.

Example : n=5 >> [5,4,3,2,1]



## 풀이

{% highlight JavaScript linenos %}

const reverseSeq = n => {
  var result = [];

  for (var i = 0; i < n; i++) {
    result.push(n - i);
  }

  return result;
};

{% endhighlight %}
